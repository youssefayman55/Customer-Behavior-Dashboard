# Customer Shopping Behavior Analysis Dashboard
📌 Project Overview

-- This project focuses on analyzing customer shopping behavior using Python, PostgreSQL (SQL), and Power BI to extract meaningful business insights from customer purchasing data.

The dashboard helps businesses understand:

Customer demographics
Purchasing behavior
Revenue trends
Product performance
Customer loyalty
Subscription impact
Shipping and payment preferences

The project combines:

Data Cleaning & Feature Engineering (Python)
Database Management & Business Queries (PostgreSQL)
Interactive Data Visualization (Power BI)
🚀 Technologies Used
Python
PostgreSQL
pgAdmin
Microsoft Power BI
Pandas
SQLAlchemy
psycopg2
📂 Dataset Information

The dataset contains customer shopping behavior data including:

Column	Description
Customer ID	Unique customer identifier
Age	Customer age
Gender	Male/Female
Item Purchased	Purchased product
Category	Product category
Purchase Amount	Purchase value in USD
Location	Customer location
Size	Product size
Color	Product color
Season	Purchase season
Review Rating	Customer rating
Subscription Status	Subscription membership
Shipping Type	Shipping method
Discount Applied	Discount usage
Previous Purchases	Customer loyalty indicator
Payment Method	Payment method
Frequency of Purchases	Purchase frequency
